# diabetes-logbook 


Logbook steps:

Steps to run Server

1. clone or download the repo, cd into /server/ folder.
2. "npm run dev"
3. Server will be running on "5000" port.

Steps to run Client

1. clone or download the repo, cd into /client/ folder.
2. "npm start"
3. Server will be running on "4200" port.
